Development with C has Six Stages:


Editing: Writing the source code by using some IDE or editor.

Preprocessing: In Preprocessing, source code is expanded into a larger form that is simpler for the compiler to understand,Preprocessor directives are always initialized at the beginning of the program. it begins with the symbol (#) hash.

Compiling: This process Creates object code and stores on disk.

Linking: linker is used to bind together all the object files and the required libraries, in order to produce an executable file.

Load: Puts Program in a Memory.

Execute: Takes each instruction in a program and executes it storing new data values.
