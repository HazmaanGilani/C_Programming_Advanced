In Preprocessing, source code is expanded into a larger form that is simpler for the compiler to understand,Preprocessor directives are always initialized at the beginning of the program. it begins with the symbol (#) hash.

"Commands used in preprocessor are called preprocessor directives" . Below is the list of preprocessor directives that C language offers.

Macro
It is used in defining Constant Symbol.
syntax:
#define

Example:
             #define size 10

It replaces the instances of size with constant value 10.

Header file inclusion
The preprocessor directive include< file name> can be used to include any file into user program if the function or macro definitions are present in the source code. syntax:
#include <file_name>



Conditional compilation

These are the Set of commands that are included or excluded in source program to Compile the Program with respect to the condition.

#ifdef, #if, #endif, #else, #ifndef

#ifdefreturns True if the macro is defined
#endifEnds preprocessor conditional
#ifchecks if a compile time condition is true
#elseThe alternative of #if
#ifndefreturns True if the macro is not defined

Other directives
#undef is used to undefine a defined macro variable.

#undef

#pragma is used to call a function before the main function and call another function after the main function in a C program

syntax:
#pragma
 
E.X:
   #pragma startup f1 #pragma exit f2

Here f1 is called before main and f2 is called after main
