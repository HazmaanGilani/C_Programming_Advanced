Operator: An operator is a symbol that tells the Computer to perform certain mathematical or logical manipulations.


Expression: An expression is a sequence or combination of operands and operators that reduces to single value
Eg: 10+20 is an expression whose value is 30
C operators can be classified into a no. of categories.

1. Arithmetic
2. Relational
3. Logical
4. Assignment
5. Increment and Decrement
6. Conditional
7. Bitwise
8. Special

Arithmetic Operators: C provides all the basic arithmetic operators
Eg: a + b, a - b, a * b,

-a * b, a / b, a % b

Here a and b are variables and are known as operands and +,-,*,/ and % are the operators.
Note:C does not have an operator for exponentiation.


Operator Arithmetic:

Integer Arithmetic: When the operands in an expression are integers then the expression is an integer expression and the operation is called integer arithmetic. This always yields an integer value. 
For Eg. a = 10 and b = 5 then

a+b=15

a-b=5

a*b=50

a\b=2

a%b=0(% is modulo operator returns remainder of expression)


Real Arithmetic / Floating Pont Arithmetic: Floating Point Arithmetic involves only real operands of decimal or exponential notation. If x, y & z are floats,
then 
x = 6.0/7.0 = 0.857143

y = -1.0/3.0 = 0.333333 

z = 3.0/2.0 = 1.500000


 Note:(Modulo Operator) % cannot be used with real operands


Mixed mode Arithmetic: When one of the operands is real and the other is integer the expression is a mixed mode arithmetic expression. 
 Eg: 

 15/10.0 = 1.500000 

 15/10 = 1

Relational Operator: These are the operators used to Compare arithmetic, logical and character expressions.the value of a relational express is either one or zero .it is 1 if one is the specified relation is true and zero if it is false

eg: 10 < 20 is true
20<10 is false

Logical operator : Logical Operators are used when we want to test more than one condition and make decisions. here the operands can be constants, variables and expressions Logical operators are &&, ||, !

Assignment Operator: Used to assign the result of an expression to a variable. = is the assignment operator.
E.g:

x=2;

x + = 1 ; == > x = x+1

Eg : 
++x   or   x ++   == >   x+=1    == > x=x+1 .

A Prefix operator first adds 1 to the operand and then the result is assigned to the variable on left. A Postfix operator first assigns the value to the variable on the left and the increments the operand.
e.g

a=5;         a=5;
x=++a         x=a++
returns:     returns:

Variable = (condition)? Value 1 : Value 2:

If the Value of the condition is true then Value 1 is e valued assigned to the varable, otherwise Value2. 
Eg: big = ( a > b ) ? a : b ; 

 This expression is equal to 

 if (a > b) 

 big = a; 

 else 

 big = b;
