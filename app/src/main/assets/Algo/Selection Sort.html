Selection Sort:
In Selection Sort we put the smallest item at the start of the list, then the next smallest item at the second position in the list, and so on until the list is in order.

Algorithm for Selection Sort:

for i=0 to N-2 {

set smallest = i

for j=i+1 to N-1 
{

compare list[smallest] to list[j]
if list[smallest] == list[j]
smallest = j

}

swap list[i] and list[smallest]

}

Time Complexity:

Best case : O(n^2)
Average case : O(n^2)
Worst case : O(n^2)
