Bubble Sort:
The bubble sort is similar to selection sort,In buuble sort algorithm,on each pass we place at least one item in its proper location.bubble sort works by comparing two items in the list at a time. bubble sort will always compare two consecutive items in the list, and swap them if they are out of order. If we assume that we start at the beginning of the list, this means that at each pass through the algorithm, the largest remaining item in the list will be placed at its proper location in the list.

Algorithm for bubble sort
for i=N-1 to 2 {

set swap flag to false

for j=1 to i {

if list[j-1] > list[j]
swap list[j-1] and list[j]
set swap flag to true

}

if swap flag is false, break. The list is sorted.

}
Time Complexity:
The complexity of sorting algorithm is depends upon the number of comparisons that are made. Total comparisons in Bubble sort is: n ( n − 1) / 2 = n^2 − n

Best case : O (n^2)
Average case : O (n^2)
Worst case : O (n^2)
