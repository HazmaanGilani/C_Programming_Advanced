Kruskal' Algorithm

Kruskal's Algorithm is a greedy algorithm. A greedy algorithm chooses some local optimum (i.e. picking an edge with the least weight in a MST). Kruskal's algorithm works as follows: Take a graph with 'n' vertices, keep on adding the shortest (least cost) edge, while avoiding the creation of cycles, until (n - 1) edges have been added. Sometimes two or more edges may have the same cost. The order in which the edges are chosen, in this case, does not matter. Different MST's may result, but they will all have the same total cost, which will always be the minimum cost.

Algorithm:

Kruskal(G,w)

 A = 0

 For each vertex v ∈ V[G]

 do make_set(v)

 sort the edge of E into nondecreasing order of weight

 For each edge(u,v) ∈ E taken in nondecresing order of weight

 do if find_set(u) ≠ find-set(v)

 then A =A U {(u,v)}

 union(u,v)

 Return A
