Algorithm for Evaluation of Postfix Expression:
clear the stack
symb = next input character
while(not end of input)
{
if (symb is an operand)
push onto the stack;
else
{
pop two operands from the stack;
result=op1 symb op2;
push result onto the stack;
}
symb = next input character;
}
return (pop (stack));
