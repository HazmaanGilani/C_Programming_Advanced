DataTypes: DataType can be defined as the type of variable or constant to store
when we use a variable in a program,we have to mention the type of data that a variable can store This can be handled using datatypes in c

There are 2 types of datatypes in c
1.Primitive DataType
2.Secondry or Derrived Datatype

Primitive DataType:Primitive types are the most basic data types C has four fundamental data types: Character, Integer, Floating-Point, Double floating-Point,These are denoted as char, int, float double respectively.

Secondry or Derrived Datatype:Derived data types are those that are defined in terms of primitive data types E.g:Array,Pointer,Strucure,Union etc


Modifiers: Modifiers are used to alter the meaning of basic data types to fit various needs.
Data type modifiers are classified into following types.

Long
Short
Signed
UnSigned
Example:
Long can be used to increase the size of the current data type to 2 more bytes, which can be applied on int or double data types. For example int occupy 2 byte of memory if we use long with integer variable then it occupy 4 byte of memory.
