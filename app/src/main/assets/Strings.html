Strings

  • In C language, an array of characters is known as a string.
    char st[80];
  • This statement declares a string array with 80 characters.
  • The control character "0"which represents a null character is placed automatically at the end of any string used. STRING HANDLING FUNCTIONS IN C:

    • There are four important string Handling functions in C language.

    1. strlen( ) function
    2. strcpy( ) function
    3. strcat( ) function
    4. strcmp( ) function
    5. strlen( ) Function:

    strlen( ) function : It is used to find the length of a character string. 
     Ex: int n; 
 char st[20] = "Programming"; 
 n = strlen(st);


  • This will return the length of the string 11 which is assigned to an integer variable n.
  •  Note: the null charcter "\0" available at the end of a string is not counted.

    • strcpy( ) Function: strcpy( ) function is used to copy from one string to another string. 
    Ex : 
 char str[15];
 strcpy(str, "Programming") ;

    This will assign the string "Programming" to the character variable str. 
     
* Note: character value like str = "Programming"; cannot be assigned in C language.

    strcat( ) Function: strcat( ) function is used to join character, Strings. When two character strings are joined, it is referred as concatenation of strings. 

     Ex: 
    
 char str[20] = "Programming";
    
 char str2[20]= "is Fun"
    
 strcat(str,str2);
  • This will join the two strings and store the result in str and the result is Programming is fun
     
  •  Note: the resulting string is always stored in the left side string variable.

    • strcmp( ) Function: strcmp ( ) function is used to compare two character strings.
  • It returns a 0 when two strings are identical. Otherwise it returns a numerical value which is the different in ASCII values of the first mismatching character of the strings being compared.
    char str1[20] = "SAD";
    
char str2[20] = "SAND";
    
strcmp(str1, str2);


  • This will return an integer value "- 10" which is the difference in the ASCII values of the first mismatching letters "D" and "N"
    * Note: the integer value obtained as the difference may be assigned to an
integer variable as follows:

    int n;
    
n = strcmp(str1, str2);

    • Additional String Handling Functions:

    Some "C" compilers will accept the following string handling functions which are available in header files string.h and ctype.h Functions:

    (i) strupr( ): to convert all alphabets in a string to upper case letters.
    Ex: strupr(" code ")  ---> " CODE"
    (ii) strlwr( ): To convert all alphabets in a string to lower case letters.
    Ex: strlwr(" CODE ")  ---> "code"
    (iii)strrev( ): To reverse a string
    Ex: strrev(" CODE ")  ---> "EDOC"
    (iv) strncmp( ): To compare first n characters of two strings
    . 
    Ex: m = strncmp (" Jony ", " Joey ", 3); ---> m = 9
    (v) strcmpi( ): To compare two strings with case in sensitive (neglecting upper / lower case)
    Ex: m=strcmpi(" VEGAS ", " vegas "); ---> m = 0.
    (vi) strncat( ): To join specific number of letters to another string.
    Ex. char s1[10] = "Las"; char s2[10] = "Vegas"; strncat(s1,s2,3);  s1 will be "LasVeg".
